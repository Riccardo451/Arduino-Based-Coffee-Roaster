# Arduino-Based-Coffee-Roaster

Based on the work done by: http://www.sinobi.dk/henrik/coffeeroaster1/

Original code at: http://www.sinobi.dk/henrik/kafferister1/cofferoaster.ino

Used libraries:
 https://github.com/adafruit/MAX6675-library
 https://github.com/smarmengol/Modbus-Master-Slave-for-Arduino

Hardware used:
- Arduino Pro mini
- MAX6675 sensor + k thermocouple
- Mosfet based PWM Fan controller
- SSR 40A
- Popcorn machine with removed thermal cutout
